## HRMS-Frappe


### Site Creation

```
▶ bench new-site hrms.site.domain
```
### Get required app

```
▶ bench get-app erpnext
▶ bench get-app hrms
```
### Installing erpnext and education

```
▶ bench --site hrms.site.domain install-app erpnext
▶ bench --site hrms.site.domain install-app hrms
```

### Start application

```
▶ bench start
```
